# donder-release

Quickly create releases on Github from the command line or CI using conventional commits.

## Usage

#### With Cargo

```bash
cargo install donder-release
```

See CLI options:

```bash
donder-release --help
```

#### With Npm

See CLI options:

```bash
npx donder-release --help
```
